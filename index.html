#ubuntu 搭建代理服务器
@(唐焰)[ununtu][代理服务器]

squid3:Squid是全功能的HTTP/1.0代理服务器，也近乎完整的兼容HTTP/1.1。Squid提供丰富的访问控制、认证和日志环境用以网站代理服务器和内容服务应用.Squid3可用于HTTP，HTTPS，FTP等网站的缓存代理服务器。

1.安装squid3
ubuntu命令行键入: sudo apt-get install squid

2.修改配置文件

  • 配置文件：/etc/squid3/squid.conf
    将第1060行的“http_access deny all” 改成“http_access allow all”
  • 修改端口 在1460 行 http_port 的3128端口号改为自己需要的端口号，3128为缺省的端口号
  • 重启squid3
    sudo service squid3 restart

3.利用代理服务器上网
在自己电脑的浏览器中把相应代理服务器地址和端口改为制作代理服务器的ip地址和自己设定的端口号 google浏览器代理服务器设置
